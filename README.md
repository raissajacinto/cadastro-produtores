# Cadastro de Produtores Culturais
Este formulario html busca recolher dados para o seu cadastro como Produtor Cultural.

## Campos para preencher:
- Produtor Cultural
- RG
- CPF
- Inscrição Municipal
- Endereço (Bairro, Cidade, Estado e CEP)
- Telefone
- Fax
- E-mail
- Profissão
- Atividade Proficional
- CNPJ
- N° de Sócios
- Inscrição Estadual
- Nome
- Data da Nomeação
- CRC
